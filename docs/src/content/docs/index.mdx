## What you get

<CardGrid>
  <Card title="Rust authoring" icon="pencil">
    Use familiar Rust syntax, crates.io packages, and `cargo check` for type
    checking - including proc macros for events, settings, and locale.
  </Card>
  <Card title="Factorio Lua output" icon="document">
    The CLI emits a normal Factorio mod directory (`info.json`, stage entry
    files, `lua/`, optional `locale/`).
  </Card>
  <Card title="Install and run" icon="rocket">
    Build into `dist/`, copy into your Factorio mods folder, and optionally
    launch the game with `factorio-rs open`.
  </Card>
  <Card title="Guides" icon="open-book">
    See [Installation](installation/), [Getting started](guides/getting-started/),
    [Language support](guides/language/), [Lints](guides/lints/), and the
    [CLI reference](reference/cli/).
  </Card>
</CardGrid>
