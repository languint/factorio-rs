## Documentation roots

Each root has its own sidebar. Add new ecosystem crates under
`docs/src/content/docs/ecosystem/<crate>/` and register them in
`docs/src/projects.ts`.

## What factorio-rs gives you

<CardGrid>
  <Card title="Typed Factorio APIs" icon="pencil">
    Write a Factorio-oriented Rust subset. `cargo check` and rust-analyzer use
    generated stubs; the CLI emits a normal Lua mod.
  </Card>
  <Card title="In-game tests" icon="approve-check">
    Author `#[test]` simulations and run them with
    [`factorio-rs test`](guides/testing/) inside Factorio (headless by default).
  </Card>
  <Card title="Cross-mod APIs" icon="add-document">
    Export functions with `#[factorio_rs::export]` and depend on other mods like
    Cargo crates - see [Share an API](recipes/share-api/).
  </Card>
  <Card title="Safety lints" icon="warning">
    Transpile-time [lints](guides/lints/) catch Rust patterns that type-check
    but nil-crash or miscompile in Lua.
  </Card>
</CardGrid>
